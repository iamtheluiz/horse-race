## 🏃 Getting Started

First, clone this respository and access the created folder:

```bash
# Cloning repository
git clone https://github.com/iamtheluiz/horse-race.git

cd horse-race/
```

Getting into the folder, install project dependencies:

```bash
npm install
# or
yarn install
```

## 👨🏽‍💻 Usage

```bash
yarn run start
```

## ⚙️ Build

To build this project, run:

```bash
yarn build
```
